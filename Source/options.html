What is HotkeyBB?

HotkeyBB is intended for usage in forum/blog/guestbook reply fields. With HotkeyBB you can insert BB-code tags as fast as lightning not depending on BB-code buttons provided. You may simply insert BB-code tag pair or surround the selected text. Complicated tags with option (like [tag="option"]text[/tag]) are also supported. Depending on tag properties, you'll be prompted for option value or the selected text will be used as option.

HotkeyBB also supports URL-based site-specific options. You may adjust some tag insertion options according to the site requirements. Available options include: quote mode for tag option, tag bracket style (BB-code or HTML) and tag case.

Warning! Extension intersepts ALL shortcuts including standard ones, so beware of defining such combinations as Ctrl-Z, Ctrl-X, Ctrl-C, Ctrl-V, Ctrl-A. However this happens only when input focus is within a textarea; when it isn't, shortcuts are working as usual.

Site options

You may adjust some tag insertion options according to the web engine requirements.

  • Quotes — whether tag options should be surrounded by quotes
  • HTML tag — use HTML-style tags ( <> ) instead of BB-code ( [] )
  • Tag upper case — insert tags in upper case

Tag list

Here's the list of all tags available. You may delete any of them and add new ones.

  • Open tag — opening tag
  • Close tag — closing tag; could be the same as opening, null (like [tag="option"]) or custom
  • Option — tag has an option (like [tag="option"]text[/tag])
  • Sel to option — selected text would be used as option
  • Hotkey — key combination which will insert the tag
